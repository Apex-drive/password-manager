# Password Manager

**Статус:** В разработке

# О проекте
Простое и безопасное приложение для хранения паролей. Все пароли хранятся в зашифрованном виде. Доступ к ним — через один мастер-пароль.

# Возможности
- Хранение паролей (сайт, логин, пароль)
- Шифрование всех данных
- Копирование пароля в буфер обмена
- Поиск по сайтам и логинам
- Удаление ненужных записей
